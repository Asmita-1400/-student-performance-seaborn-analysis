# 📊 Student Performance Data Visualization using Seaborn

This project analyzes a **Student Performance dataset** using Python libraries like **Pandas, Seaborn, and Matplotlib**.  
The goal is to visualize different student score patterns and understand relationships between variables.

---

## 🚀 Technologies Used

- Python
- Pandas
- Seaborn
- Matplotlib

---

## 📂 Dataset

The dataset used in this project is:

StudentsPerformance.csv

It contains information about student scores in:

- Math
- Reading
- Writing
- Gender
- Race/Ethnicity
- Parental Education
- Lunch Type
- Test Preparation Course

---

## 📊 Visualizations Created

This project generates multiple visualizations such as:

1. Distribution of Math Scores
2. Gender-wise Score Comparison
3. Reading vs Writing Score Relationship
4. Score Distribution using Histograms
5. Box Plot Analysis of Student Scores

These visualizations help understand how different factors influence student performance.
